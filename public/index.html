How to Use the Searches

Similar Meaning Search: Enter a brief description of a concept or word you can’t recall. Example: “ringing in the ears” might return tinnitus.

Sounds Like Search: Type in how the word sounds, even if you’re not sure of the spelling. Example: “nife” could return knife.

Adjective Search: Enter a noun to find adjectives commonly used to describe it. Example: Enter “ocean” and get vast, salty, blue.

Noun Search: Enter an adjective to find nouns commonly described with it. Example: Enter “fluffy” and get pillow, bunny, cloud.

Define Search: Enter a word to search for that words definition. Example: Enter "acouasm" and get The perception of ringing, buzzing or hissing sounds that are not really there.
